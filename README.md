# 8051-thermostat

This is a school project for the microcontrollers course, it's a simple thermostat that has an LCD display, temperature adjusting buttons and turns on an external relay. 

It is battery powered and uses the AT89C2051 MCU, it features an LM85 temperature sensor, a boost DC-DC converter and a shift register to expand the output pins

The board itself is a 4 layer board with SIG/GND/PWR/SIG stackup.
